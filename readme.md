## Установка

```
git clone https://github.com/FoxSeaSha/Daac-Gallery.git foxve.loc
composer install
php artisan key:generate
php artisan migrate
php artisan db:seed
```

## Функционал

```
1. Свободная регистрация
2. Авто изменения размеров изображений до максимум FullHD при загрузки на сервер.
3. Введение нескольких категорий  изображений
4. Разделение на публичные категорий и публичные изображения
5. Аккаунт админа который может убирать и менять изображения в категории, а также подтверждать регистрацию пользователей или банить пользователей, сортировать по последним добавлениям от конкретного пользователя /от всех пользователей
6. Система комментариев к изображениям
7. Фильтрация спама в комментариях.
```
